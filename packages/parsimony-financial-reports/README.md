# parsimony-financial-reports

[FinancialReports.eu](https://financialreports.eu/) connectors for
[parsimony](https://parsimony.dev). Wraps the official client to expose
companies search, filings, markdown rendering, ISIC browsing, and ISIN lookup.

## Install

```bash
pip install parsimony-financial-reports
```

The provider is registered via the `parsimony.providers` entry-point group.

## Required environment

```bash
export FINANCIAL_REPORTS_API_KEY=...
```
